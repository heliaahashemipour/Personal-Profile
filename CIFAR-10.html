Project Summery:

One of the applications of neural networks is classification. In this project, we are going to categorize images. We want to use different neural networks (specifically fully networks connected) Let's build a model that takes photos as input and recognizes the category of each photo. In this problem, we want to classify the images of the CIFAR-10 dataset. This dataset is a smaller version of the CIFAR-100 dataset, limited to images from 10 different categories. To reduce the computational complexity, we only use the images of the first four classes of this dataset. The dimensions of the images in this dataset are equal to 32 (length) by 32 (width) by 3 (RGB color channels).
